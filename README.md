# polyglot

A Leiningen template for combined Java/Clojure projects.

## Usage

### Testing

      chmod +x ./run-{{simple-name}}.sh
      ./run-{{simple-name}}.sh

#### Clojure tests

      lein test
      
#### Java tests

      lein run-junit
